Add beautiful GAP-style Gsap Animations to Elementor containers with scroll-triggered effects.

== Description ==

This plugin adds professional Gsap Animations to Elementor containers with scroll-triggered effects powered by GSAP.

= Features =
- Multiple animation types (reveal me, text appear, reveal text)
- Customizable duration and delay
- Scroll-triggered animations
- Lightweight and performant
- Works with Elementor containers

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plg-gsap-animations` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to any Elementor container and find the new "Gsap Animation" tab in the style settings

== Frequently Asked Questions ==

= Does this require GSAP? =
Yes, but it's bundled with the plugin so you don't need to install it separately.

= Can I use this with sections and widgets? =
Currently it only works with Elementor containers, but we may expand to other elements in future versions.
